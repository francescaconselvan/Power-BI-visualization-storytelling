## Datasets

This repository includes three diverse datasets suitable for practicing Power BI visualizations and storytelling:

### 1. **Bike Sales Data (2013-2023)**
- **File:** `bike_sales_data_world_2013_2023.csv`
- **Size:** ~50 MB
- **Period:** 2013-2023
- **Use Cases:** Time series analysis, sales trends, global market analysis, seasonal patterns

### 2. **BMW Sales Data (2010-2024)**
- **File:** `BMW sales data (2010-2024) (1).csv`
- **Size:** ~3.4 MB
- **Period:** 2010-2024
- **Use Cases:** Automotive industry analysis, long-term trend analysis, brand performance

### 3. **Most Streamed Spotify Songs 2024**
- **File:** `Most Streamed Spotify Songs 2024.csv`
- **Size:** ~1.1 MB
- **Period:** 2024
- **Use Cases:** Music industry analytics, popularity metrics, contemporary data visualization

All datasets are located in the [datasets](datasets/) directory and ready to use with Power BI Desktop.

## Getting Started

### Prerequisites
- [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (latest version recommended)
- Basic understanding of data analysis concepts
- Familiarity with CSV file formats

### Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/Power-BI-visualization-storytelling.git
   cd Power-BI-visualization-storytelling
   ```

2. **Open Power BI Desktop**

3. **Import a dataset:**
   - Go to Home > Get Data > Text/CSV
   - Navigate to the `datasets/` folder
   - Select one of the available CSV files
   - Click "Load" or "Transform Data" to begin

4. **Start creating visualizations:**
   - Explore the data in the Fields pane
   - Drag fields onto the canvas to create visualizations
   - Apply the principles from the recommended resources

### Recommended Learning Path

1. **Start with fundamentals:** Complete the DataCamp Introduction to Power BI course
2. **Read core concepts:** Begin with "Storytelling with Data" by Cole Nussbaumer Knaflic
3. **Practice with datasets:** Use the provided datasets to create your first dashboards
4. **Deepen knowledge:** Explore the Microsoft certifications and additional books
5. **Apply best practices:** Reference "The Big Book of Dashboards" for real-world patterns
