# Resources

This folder contains curated learning resources for mastering Power BI, data visualization, and storytelling with data.

## Contents

### [Books](books.md)
A comprehensive collection of recommended books covering:
- **General Overview** - Foundational texts on data visualization principles and storytelling (Knaflic, Tufte, Cleveland)
- **Technical** - Practical Power BI implementation and dashboard design guides
- **Graphic Design** - Historical perspectives and creative approaches to information design

### [MOOC Courses](MOOC.md)
Online courses and certifications for Power BI:
- **DataCamp Introduction to Power BI** - Quick 3-hour crash course covering essential fundamentals
- **Microsoft Power BI Data Analyst Professional Certificate** - Comprehensive certification program from Microsoft on Coursera
- **Microsoft Data Visualization Professional Certificate** - Specialized courses on report design, formatting, and creative design in Power BI
