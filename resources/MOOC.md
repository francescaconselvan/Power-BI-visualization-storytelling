# Useful MOOC Courses on Power BI
---

## Data Camp Introduction to Power BI  
**Provider:** Datacamp  
**Link:** https://app.datacamp.com/learn/courses/introduction-to-power-bi   
**Content:** This is an excellent crash course that covers all essential Power BI fundamentals in just 3 hours. It takes you through the complete workflow from data loading to creating interactive reports.

---
## Microsoft Power BI Data Analyst Professional Certificate (MICROSOFT - Coursera)  
**Course:** Data Visualization Fundamentals  
**Link:** https://www.coursera.org/learn/data-visualization-fundamentals/home  
**Content:** This course introduces the fundamentals of data visualization through the basics of creating visually appealing and meaningful reports, charts and graphs. 

---

## Microsoft Data Visualization Professional Certificate (Microsoft - Coursera)
**Course 1:**  Data Analysis and Visualization with Power BI
**Link:** https://www.coursera.org/learn/data-analysis-and-visualization-with-power-bi?specialization=microsoft-power-bi-data-analyst
**Content:** This course teaches report design and formatting in Power BI, covering how to create visuals for reports and dashboards, and use report navigation for data-driven storytelling  

**Course 2:** Creative Designing in Power BI  
**Link:** https://www.coursera.org/learn/creative-designing-in-power-bi?specialization=microsoft-power-bi-data-analyst
**Content:** This course focuses on creating compelling and cohesive reports and dashboards through effective design principles.
